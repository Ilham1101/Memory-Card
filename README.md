# 🃏 Memory Card Quiz

**Memory Card Quiz** adalah sebuah proyek game kuis interaktif yang menggabungkan konsep *memory card* dengan sistem pengelolaan pertanyaan. Pemain dapat menguji kemampuan mengingat sekaligus menjawab berbagai pertanyaan yang telah disediakan dalam permainan.

Proyek ini dirancang agar pertanyaan dapat disimpan dan dikelola dengan mudah, sehingga koleksi soal dapat terus diperbarui, ditambah, atau dimodifikasi sesuai kebutuhan. Dengan desain yang sederhana namun interaktif, aplikasi ini cocok digunakan sebagai media pembelajaran, hiburan, maupun contoh implementasi pengelolaan data dalam pengembangan aplikasi.

Melalui proyek ini, pengguna dapat merasakan pengalaman bermain yang menyenangkan sekaligus mengasah daya ingat dan pengetahuan melalui berbagai jenis pertanyaan yang tersimpan di dalam sistem.
